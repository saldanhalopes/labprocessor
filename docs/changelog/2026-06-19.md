## Import: Dados MFVCQ Implementados

### O que é
Sistema de análise de Controle de Qualidade farmacêutico integrado ao LabProcessor.
Baseado em duas planilhas extraídas (BASEFLUXO + MFVCQ Demanda).

### Endpoints criados
| Rota | Descrição |
|------|-----------|
| `POST /api/mfvcq/analyze` | Analisa produto: gera fluxo CQ + demanda |
| `GET /api/mfvcq/search?q=` | Busca produtos na base de demanda |
| `GET /api/mfvcq/indices` | Índices de busca (ativos, formas, rotas...) |
| `GET /api/mfvcq/template` | Template para novo produto |

### Dados carregados
- `basefluxo_estruturado.json` — Fluxos hierarquizados de CQ
- `demanda_estruturada.json` — 1.571 produtos com demanda
- `indices_busca.json` — Índices para busca rápida
- `template_novo_produto.json` — Template para novos produtos
- `analyze_product.py` — Script Python original preservado

### Como usar
```bash
# Analisar um produto
curl -X POST http://localhost:8080/api/mfvcq/analyze \
  -H 'Content-Type: application/json' \
  -d '{"ativo":"ABIRATERONA","forma":"Sólidos","mediaMensal":10000,"fatorConversao":2,"tamanhoBulk":500}'

# Buscar produtos
curl http://localhost:8080/api/mfvcq/search?q=losartana
```
